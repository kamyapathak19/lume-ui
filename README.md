# LUME ✨

LUME is a futuristic smart projection system interface built using Next.js.

## 🚀 Live Demo
https://kamyapathak19.github.io/lume-ui/

## 💡 Features
- Theme-based experience (sleep, gaming, meditation, etc.)
- Remote control UI
- Brightness and sound adjustment
- Auto mode toggle
- Smooth futuristic UI

## 🛠 Tech Stack
- Next.js
- React
- Tailwind CSS
- Vercel (deployment)

## 📂 How to Run Locally
npm install
npm run dev
